# Student Grade Tracker (Java)

This is a Java console-based application to manage student grades efficiently.

## Features
- Add multiple student marks
- Calculate average score
- Find highest score
- Find lowest score
- Display complete summary report

## Concepts Used
- Arrays / ArrayList
- Loops
- Conditional statements
- Basic OOP concepts

## Output
Displays student performance report in console.

## Purpose
Helps in understanding basic data handling and logic building in Java.
